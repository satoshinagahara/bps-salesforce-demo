# P.1  製品概要

## 1.1 機種位置づけ

A-1000シリーズは、BPS Corporationが**中〜大型陸上風力発電サイト向け**に展開するフラッグシップ機である。2018年の市場投入以来、国内外で200基以上の納入実績を持ち、特に**内陸部の平均風速7〜9m/sクラス**の案件において高い信頼性を示している。

## 1.2 主要諸元

| 項目 | 値 | 備考 |
|---|---|---|
| 定格出力 | 5,000 kW | 定格風速時 |
| ローター径 | 150 m | 3枚翼 |
| スイープ面積 | 17,671 m² | — |
| ハブ高さ | 110 m / 125 m | 2タイプ選択可 |
| カットイン風速 | **3.5 m/s** | ブレード回転開始 |

| 発電機形式 | 永久磁石同期型（PMSG） | ギアレス |
| 周波数 | 50 / 60 Hz | 両対応 |

| 耐風クラス | IEC 61400-1 Class IIA | — |

## 1.3 市場セグメント

A-1000は以下の市場セグメントをターゲットとする：

- **主要ターゲット**: 平均風速7〜9 m/sの**優良立地**（内陸山間部、沿岸部）
- **副次ターゲット**: 平均風速6〜7 m/sの標準立地
- **非推奨**: 平均風速6 m/s未満の低風速立地（経済性の観点から非推奨）

<div class="note">

**📝 営業向け注記**: 平均風速5 m/s以下の案件が増加傾向にあるが、A-1000シリーズは元来優良立地向けに最適化されており、**低風速域での経済性確保は設計上の課題**として認識されている。該当案件については製品設計部と個別協議のこと。

## 2.2 制御系統アーキテクチャ

A-1000の制御系統は3階層構成をとる：

1. **ローカル制御**（各機構固有）: ピッチ制御、ヨー制御、コンバータ制御
2. **タービン統合制御**（SCADA Level 1）: 運転モード管理、異常検知、出力調整
3. **遠隔監視**（SCADA Level 2）: BPS中央監視センターからの遠隔診断・ファーム全体最適化

**ブレードピッチ制御機構**は、ローター系統の中でも特に**発電効率と負荷バランスを司る中核部品**である。次ページで制御仕様の詳細を記載する。

# P.3  制御仕様（ブレードピッチ制御）

## 3.1 設計思想

ブレードピッチ制御は、風況に応じてブレード角度を動的に調整し、以下の目的を達成する：

- **発電効率の最大化**: 風速に対する最適翼角度を維持
- **機械負荷の低減**: 突風時のローター・ドライブトレイン負荷を抑制
- **出力制御**: 定格風速超過時の出力平滑化

A-1000のピッチ制御アルゴリズムは、BPSが長年蓄積した運用データを基に最適化されており、**定格風速12 m/s前後での発電効率を最大化**することを第一目標としている。

## 3.2 風速域別の制御モード

A-1000のピッチ制御は風速域に応じて4つのモードを切り替える：

| モード | 風速域 | 制御目標 |
|---|---|---|
| **停止モード** | 〜3.5 m/s | ブレード回転停止（フェザリング） |
| **起動モード** | 3.5〜**5.0 m/s** | 起動トルク確保のみ。発電効率は最適化対象外 |
| **部分負荷モード** | **5.0〜12.0 m/s** | 発電効率最大化（最適翼角度追従） |
| **定格負荷モード** | 12.0〜25.0 m/s | 出力平滑化（定格出力維持） |

<div class="warn">

**⚠️ 制御設計上の前提**: 現行のピッチ制御アルゴリズム（v3.2）は、**風速5.0 m/s以上の運転域を発電効率最適化の対象**としている。これは、A-1000の想定立地（平均風速7〜9 m/sクラス）では5 m/s未満の運転時間が全体の15%未満に留まることを前提に、**制御パラメータの収束速度と演算負荷のバランスを優先した設計判断**による。

したがって、**風速3.5〜5.0 m/sの低風速域（起動モード）では発電効率は最適化されておらず**、この領域での発電量改善は現行アルゴリズムのスコープ外である。

</div>

## 3.3 最適化アルゴリズム詳細

部分負荷モード（5.0〜12.0 m/s）では、以下のアルゴリズムにより最適翼角度 θ_opt を算出する：

- 風速・ローター回転数・発電機出力の3要素から翼端速度比（TSR）を計算
- TSR目標値（A-1000では λ_opt = 8.5）への収束制御を実施
- 収束速度は500 ms以内、演算周期は100 ms

**アクチュエータ構成**

- 各ブレードに独立した電動ピッチモータ（油圧レス）
- 配置位置: ブレード根元部（ハブ内蔵）
- 応答速度: 最大 8°/秒

## 3.4 既知の設計課題（内部共有のみ）

<div class="known-issue">

**🔍 製品設計部 技術メモ（2024年12月更新）**

営業部門から、**平均風速5 m/s以下の低風速立地案件**の問い合わせが増加傾向にあるとの報告を受けている。本機の起動モード（3.5〜5.0 m/s）は現行アルゴリズムでは発電効率最適化の対象外であり、以下の技術課題として認識している：

1. **低風速域の翼角度制御パラメータ未最適化**
2. **低回転数域でのTSR追従性能の不足**
3. **ローター起動時の慣性損失の影響が相対的に大きい**

この課題への対応は **A-1000シリーズの次期ファームウェア（v4.0候補）の検討項目** となっているが、2025年3月時点では正式な開発計画には組み込まれていない。顧客要求が顕在化した場合は個別協議のこと。

</div>
