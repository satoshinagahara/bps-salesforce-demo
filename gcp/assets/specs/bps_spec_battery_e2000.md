# P.1  製品概要

## 1.1 機種位置づけ

E-2000 EnerCharge Proシリーズは、BPS Corporationが**産業用・系統用蓄電市場**向けに展開する大容量リチウムイオン蓄電システムである。工場・商業施設のピークカット、再生可能エネルギー発電所の出力安定化、マイクログリッドのバックアップ電源として導入実績が拡大している。

## 1.2 主要諸元

| 項目 | 値 | 備考 |
|---|---|---|
| 定格容量 | 500 kWh | モジュール拡張可 |
| 定格出力 | 250 kW | 充放電双方向 |
| バッテリーセル | リン酸鉄リチウム（LFP） | 高安全性 |
| セル構成 | 16モジュール × 14直列 × 8並列 | 1,792セル |
| システム電圧 | DC 700〜900V | — |
| 充放電効率 | **92%**（DC-DC） | AC-AC換算では約88% |
| 設計寿命 | **6,000サイクル**（80% DoD） | 25℃基準 |
| 動作温度範囲 | -10℃〜+45℃ | 空冷式 |
| 冷却方式 | **強制空冷** | ファンユニット × 4 |
| 通信プロトコル | Modbus TCP / CAN bus | SCADA連携可 |
| 筐体 | 20ftコンテナ型 | 屋外設置可（IP55） |
| 重量 | 約8,500 kg | 基礎工事必要 |

## 1.3 市場セグメント

E-2000は以下の市場をターゲットとする：

- **主要ターゲット**: 工場・商業施設のピークカット／デマンドレスポンス
- **副次ターゲット**: 再エネ発電所の出力安定化（太陽光・風力併設）
- **拡大中**: マイクログリッド・EV充電ステーション併設

<div class="note">

**📝 営業向け注記**: 近年、**高温環境（東南アジア・中東）での導入問い合わせ**が増加傾向にある。E-2000の動作温度上限は+45℃だが、直射日光下のコンテナ内温度は+55℃を超える場合がある。現行の空冷システムでは**高温環境での性能維持に限界**があり、製品設計部と個別協議のこと。

</div>

## 2.2 BMS アーキテクチャ

E-2000のBMS（バッテリー管理システム）は3階層構成をとる：

1. **セルBMS**（CMU: Cell Monitoring Unit）: 各セルの電圧・温度をリアルタイム監視（1秒周期）
2. **モジュールBMS**（BMU: Battery Management Unit）: モジュール単位のSOC/SOH算出、セルバランシング
3. **システムBMS**（SBMS: System BMS）: 全体制御・PCS連携・SCADA通信・異常判定

**BMS は蓄電システムの寿命・安全性・性能を司る中核システムである**。次ページで制御仕様の詳細を記載する。

# P.3  制御仕様（BMS: バッテリー管理システム）

## 3.1 設計思想

BMSの主要機能は以下の3つ：

- **安全管理**: 過充電・過放電・過電流・過温度の検知と保護
- **寿命最大化**: セルバランシング、充放電プロファイル最適化
- **性能監視**: SOC（充電率）・SOH（健全度）のリアルタイム算出

E-2000のBMSアルゴリズムは、**常温環境（15〜35℃）での運用を前提**に最適化されている。

## 3.2 温度管理モード

BMSは環境温度に応じて4つのモードを切り替える：

| モード | 温度域 | 制御目標 |
|---|---|---|
| **低温保護モード** | 〜-10℃ | 充電禁止、放電は出力制限（50%） |
| **通常運転モード** | **-10℃〜+35℃** | 充放電効率最大化（SOCバランシング最適化） |
| **高温注意モード** | **+35℃〜+45℃** | 出力制限（75%）、冷却ファン最大回転 |
| **高温保護モード** | +45℃〜 | 充放電停止、アラーム発報 |

<div class="warn">

**⚠️ 制御設計上の前提**: 現行のBMSアルゴリズム（v2.4）は、**セル温度35℃以下を発電効率最適化の対象**としている。35〜45℃の高温注意モードでは出力を75%に制限するだけで、**高温環境に特化した充放電プロファイルの最適化は行っていない**。

これは、E-2000の想定設置環境（日本国内の工場・商業施設、年間平均気温15〜25℃）ではセル温度が35℃を超える時間が全体の10%未満に留まることを前提とした設計判断による。

したがって、**高温環境（東南アジア・中東等、外気温40℃超が常態の地域）ではバッテリー寿命が設計値（6,000サイクル）を大幅に下回るリスク**がある。

</div>

## 3.3 SOC/SOH 算出アルゴリズム

### SOC（State of Charge: 充電率）
- クーロンカウント法 + OCV（開回路電圧）補正のハイブリッド方式
- 精度: ±3%（常温時）、**±5〜8%（高温時、精度劣化）**
- 更新周期: 1秒

### SOH（State of Health: 健全度）
- 内部抵抗推定 + 容量劣化トレンド分析
- 更新周期: 充放電サイクル完了時
- **高温環境でのSOH推定精度は未検証**（設計上の制約）

## 3.4 既知の設計課題（内部共有のみ）

<div class="known-issue">

**🔍 製品設計部 技術メモ（2025年3月更新）**

東南アジア・中東市場からの引き合い増加に伴い、以下の技術課題が顕在化している：

1. **高温環境での充放電プロファイル未最適化**: 35〜45℃の高温注意モードでは出力制限のみで、温度に応じた充放電カーブの動的調整が行われていない
2. **空冷の限界**: 外気温40℃超の環境では空冷ではコンテナ内温度を35℃以下に維持できない。液冷への移行検討が必要
3. **高温環境でのSOC精度劣化**: クーロンカウント法のドリフトが高温環境で加速する。カルマンフィルタ等の適応型推定アルゴリズムの導入が候補
4. **サイクル寿命の保証値問題**: 6,000サイクル（25℃基準）の保証が高温環境では適用外。温度補正された寿命カーブの策定が未了

これらの課題への対応は **E-2000シリーズの次期ファームウェア（v3.0候補）** として検討中だが、2025年6月時点では正式な開発計画には組み込まれていない。

</div>
